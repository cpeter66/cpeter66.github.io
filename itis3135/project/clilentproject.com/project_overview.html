UNC Charlotte Photography Club

Project Overview

Application Purpose: The UNC Charlotte Photography Club website will showcase student work, promote club events, and provide resources for photographers of all skill levels. The platform will feature a gallery, event calendar, and educational materials.

Intended Users:
• Current club members
• Prospective members (UNC Charlotte students)
• Photography enthusiasts in the UNCC community
• Local photographers interested in collaborations

Website Content Overview:
• Member photo gallery with filtering options
• Event calendar for workshops/photo walks
• Resources for learning photography
• About page with club leadership info
• Membership sign-up form

Dynamic Functionality

Planned Interactive Features:

  • Gallery Lightbox (Gallery Page): Full-screen photo viewer with navigation arrows. Example: Unsplash's viewer
  • Event Calendar (Events Page): Interactive calendar with AJAX-loaded details. Example: Meetup's calendar
  • Photo Filtering (Gallery Page): Filter photos by category (portrait, landscape, etc.). Example: NatGeo's filtering
  • Resource Accordion (Resources Page): Collapsible sections for different tutorial topics. Example: Photography Life's layout
